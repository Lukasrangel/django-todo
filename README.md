# Sistema Simples de To-Do List com Django

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprender e praticar o uso do Django.

Trata-se de uma aplicação simples de lista de tarefas (To-Do List) com sistema de autenticação de usuários.

## Funcionalidades

* Cadastro e login de usuários
* Criação, edição e exclusão de tarefas
* Marcação de tarefas como concluídas
* Separação de tarefas por usuário

## Uso

O sistema foi desenvolvido para uso pessoal, mas pode ser utilizado por múltiplos usuários.

## Deploy

A aplicação está em produção em um ambiente pessoal (homelab).

## Tecnologias utilizadas

* Python
* Django
* SQLite 

## Objetivo

Este projeto tem fins educacionais, com foco no aprendizado de desenvolvimento web com Django.
